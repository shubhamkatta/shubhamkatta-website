# design.md — Shubham Katta Personal Site Theme

## Intent
This is not a recruiter-first portfolio. It is a **personal internet space**.

Professional work is present, but it is only one room in the house. The website should feel like a thoughtful, expressive, slightly playful, human archive that contains:
- writing
- personal experiences
- observations
- casual findings
- case studies
- work history

The emotional target is:
**alive, personal, memorable, internet-native, thoughtful, warm, and a little mischievous**.

Not:
- sterile
- overly polished
- corporate
- generic creator template
- premium in a lifeless way

## Tone
The voice should feel like someone who:
- thinks deeply
- writes honestly
- notices human patterns
- uses humour as a truth tool
- is self-aware without self-erasing
- can move between software, psychology, internet culture, and lived experience naturally

Preferred tone mix:
- 40% thoughtful
- 20% witty / dry
- 20% intimate / reflective
- 20% clear / structured

Avoid:
- startup bro language
- generic personal-brand clichés
- “trusted leader with proven synergies” tone
- overexplaining emotions
- self-deprecating humor that destroys credibility

## Visual System
### Overall style
Use a **scrapbook / note-card / sticker** visual language with restraint.
It should feel curated, not chaotic.

Visual references:
- scrapbook layering
- taped notes
- softly rotated paper cards
- mixed paper colors
- warm cream background
- occasional doodle energy
- modern editorial typography

### Palette
- Background: `#fff8ef`
- Text: `#1d1815`
- Red/coral accent: `#ff6b6b`
- Soft yellow: `#fff6db`
- Soft blue: `#eaf7ff`
- Soft coral: `#ffe9e4`
- Dark contrast section: `#1d1815`

Use color to create personality and rhythm, not noise.

### Shapes and spacing
- rounded cards: 28px–40px
- pill buttons and tags
- layered shadow depth
- asymmetry is welcome
- sections should breathe generously
- allow occasional small rotations on cards
- avoid rigid corporate symmetry everywhere

### Bitmoji / illustration presence
The homepage should include a **self-presence anchor**:
- bitmoji
- illustrated avatar
- or stylized portrait

This should feel like “the host of the space” rather than a headshot dropped into a template.

## Content Hierarchy
### Home page order
1. Hero with identity + chips + playful CTA
2. Personal intro / “Me”
3. Writings
4. Casual findings
5. Work
6. Contact

### Principle
**Personality first. Work second.**

The site should introduce the human before the résumé.

## Copy Rules
### Good copy feels like
- direct
- human
- observant
- occasionally surprising
- conversational without being sloppy

### Good humor looks like
- dry observations
- psychological insight
- truth stated sideways
- warmth and wit in microcopy

### Bad humor looks like
- clownish gimmicks
- random memes
- undermining competence
- trying too hard to be quirky

## Component Rules
### Cards
Cards should feel like objects placed on a desk:
- subtle tilt allowed
- mixed paper tones allowed
- each card should have a strong headline and short body
- cards should be readable even when visually expressive

### Buttons
Buttons should feel tactile, almost like stickers or labels.
Use strong weight and clear contrast.

### Section labels
Use uppercase micro-labels for orientation:
- me
- writings
- work
- case files
- say hello

### Findings block
The “findings” area can go darker and moodier.
It should feel like collected notes or observations pinned together.

## Extension Rules for Future AI Tools
When extending the site:
1. Protect the personal-first hierarchy.
2. Preserve the scrapbook / paper-note identity.
3. Prefer adding meaningful writing over decorative UI.
4. Never convert the site into a generic corporate portfolio.
5. When adding new pages, make them feel like part of the same internet home.
6. Keep work clear and credible, but don’t let it dominate the atmosphere.
7. Any new humor should sharpen truth, not flatten sincerity.

## Recommended Future Enhancements
- swap emoji with actual bitmoji / illustration
- add hand-drawn dividers or subtle doodle SVGs
- add reading-time and category chips on posts
- add a “now” page
- add a “notes / tiny thoughts” stream
- add a “journeys” section for personal experiences
- add Open Graph images matching the scrapbook theme

## Red Flags
Reject changes that make the site feel:
- too monochrome
- too minimal in a luxury-tech way
- too recruiter-optimized
- too SEO-farmed
- too shiny and soulless
- too random and visually messy

## Final Test
If the result feels like:
“a real person with taste, technical depth, emotional intelligence, and an internet personality built this”
then it is on track.
